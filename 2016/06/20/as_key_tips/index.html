Android Studio常用快捷键说明

总结了比较常用的快捷键，由于操作系统、设置的不一样可能导致默认的快捷键不一样,因此就不在下面说明
，只是列举了快捷键关键字以及说明，大家可以去Studio的keymap里自行搜索设置

Navigate Class….(Quickly navigate a class by name)

根据类名快速查找，非常好用

File Structure

快速浏览文件结构，列举当前类的方法和属性

Search Everywhere

查找任何文件

Delete Line

删除整行

Hide All Tool Windows

关闭非代码区的所有窗口,专注于敲代码

Find Usages

查找引用当前变量和方法的地方

Quick Definition

可以快速预览类或者方法的实现

Bookmarks

可以快速打开已经添加到书签的文件

Find Action

可以根据输入的关键词匹配各种动作,如debug、run、clean、跳转到plugins、设置等

Move Lines Up/Down

移动整行代码

Parameter Info

快速查看方法的参数

Recent Files

最近打开文件列表

Line(Navigator to a line)

快速跳转到指定行数

Run

运行app

Related Symbol

跳转到当前Activity、Fragment的xml文件,如果当前类里有多个布局文件,则列举出来供选择

Rename

重命名
